# 更新日志

所有项目的重要变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，

## [1.0.0] - 2026-01-08

### ✨ 新增

- **核心功能**
  - 12 个精选 LangChain 项目推荐
  - 智能筛选系统（按难度、技术标签、关键词）
  - 项目代码可用性指标（0-100%）
  - 项目完成状态标记功能
  - 学习进度追踪和统计

- **用户界面**
  - 现代化的深蓝紫色主题设计
  - 响应式布局（桌面、平板、移动）
  - 可收起的进度统计面板
  - 流畅的动画和过渡效果
  - 项目卡片展开/收起详情

- **数据管理**
  - localStorage 本地存储进度数据
  - 导出进度为 JSON 文件
  - 导入进度从 JSON 文件
  - 一键重置所有进度

- **部署和文档**
  - Docker 容器化部署配置
  - Nginx 服务器配置
  - Apache 服务器配置
  - 自动部署脚本
  - 详细的部署指南
  - 完整的项目文档

### 🎨 设计

- 深蓝紫色主题配色
- 现代化的卡片设计
- 清晰的信息层级
- 流畅的用户交互

### 🚀 技术栈

- React 19 + TypeScript 5.6
- Vite 7 构建工具
- Tailwind CSS 4
- shadcn/ui 组件库
- Framer Motion 动画
- Wouter 路由

### 📚 文档

- QUICKSTART.md - 快速开始指南
